Belangrijke design-keuzes

  • Approval-flow: nieuwe kolom review_status op job_sources (pending / approved / rejected). Default pending voor career-pages. Aggregators krijgen approved via migratie.
  • Auto-creation timing: bij run status='completed', ongeacht of Pipedrive-sync slaagt. Suggesties verschijnen op run-detail; echte rij in job_sources bestaat al maar met review_status='pending'.
  • Geparsede sitemap: data zit al in master_record.pages_crawled + website-source.parsed.discovered_urls. Nieuw blok rendert dat — geen DB-werk.
  • Forward-compat voor V1B (Inngest scraper): kolom next_scrape_at nu toevoegen. Scraper picks alleen review_status='approved' AND active=true AND next_scrape_at <= now().
  • Tab-locatie: in /companies/[id] nieuwe "Werken-bij" tab (gevraagd). NIET in run-detail — die heeft alleen approval-blok.
